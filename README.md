# 项目说明

由于版权限制，多数歌曲无法播放，建议使用林俊杰歌曲试听。

服务端采用[Martinell1/NeteaseCloudMusicApi: 网易云音乐 Node.js API service (github.com)](https://github.com/Martinell1/NeteaseCloudMusicApi)

可能随时发生变化。
